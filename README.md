# 太空控制台「边缘指令速记」

一款贴合边缘计算特性的极简记忆类小游戏，兼具碎片化娱乐性与边缘计算知识科普属性，基于阿里云 ESA 全链路技术构建，实现一键部署、轻量传播、低延迟体验。

## 🌟 核心玩法

模拟“边缘节点接收指令并快速执行”的核心特性，无复杂逻辑，轻量化娱乐：

1. 屏幕短时弹出随机生成的“边缘指令代码”（如`pages-deploy-01`、`edge-func-08`）；
2. 玩家需在限定时间内，在终端风格输入框中输入完全相同的指令；
3. 系统按**输入正确率** + **响应速度** 综合计分，适配通勤、摸鱼等碎片化场景。

## 🎨 科技风视觉设计（贴合边缘计算特性）

整体界面仿航天控制台风格，视觉体验与边缘计算技术属性深度绑定：

- **全局界面**：背景采用边缘节点分布热力图（基于 ECharts 极简科技风模板定制），还原太空/边缘计算控制台质感；
- **输入交互**：仿终端输入框设计，输入字符带霓虹闪烁效果；错误字符实时标红，并触发“指令错误”警告（红色辉光+轻微震动反馈）；
- **数据面板**：实时展示“指令正确率”“平均响应速度”，极简折线图动态更新，复刻监控大屏视觉。

## 🚀 核心应用价值（部署即用/易传播）

贴合“轻量化、易落地、高传播性”设计目标，低门槛使用+高科普价值：

1. **一键部署**：基于阿里云 Pages 制作模板，用户无需编写代码，点击“导入模板”即可一键部署专属游戏；
2. **轻量传播**：游戏整体体积＜ 1MB，全端适配（移动端/PC 端），支持分享“个人最高记录”到社交平台；
3. **知识科普**：每轮游戏结束后，弹窗展示当前指令对应的阿里云 ESA 产品小知识（如`pages-deploy`：阿里云 Pages 一键部署前端应用）。

## 🔧 ESA 技术链路（低开发成本）

全链路基于阿里云 ESA 产品构建，兼顾性能与开发效率：

| ESA 产品       | 具体用途                                                                   |
| -------------- | -------------------------------------------------------------------------- |
| 阿里云 Pages   | 托管游戏前端（纯 HTML+CSS+JS，无框架依赖，核心代码量＜ 800 行）            |
| 阿里云边缘函数 | 1. 随机生成指令字符串；2. 校验用户输入指令的正确性（就近计算，低延迟响应） |
| 阿里云边缘存储 | 存储用户最高记录、游戏次数、常用指令类型等轻量化数据                       |
| 阿里云边缘缓存 | 加速控制台背景、ECharts 图表素材、音效等静态资源加载，降低访问延迟         |
